INTR0DUCTION TO HTML

BASICS OF HTML

Tag Description
<!DOCTYPE html> Defines the document as HTML5, helping the browser interpret it correctly.
<html> The root element that contains all other HTML elements on the page.
<head> Contains meta information about the document such as title and links to CSS or scripts.
<title> Specifies the title of the document, which is shown in the browser tab.
<body> Holds all visible page content like text, images, and links.
<h1> to <h6> Used for headings; <h1> is the largest and most important, <h6> is the smallest.
<p> Defines a paragraph of text.

ELEMENTS AND TAGS

Tag Name Opening Tag Closing Tag
Paragraph <p> </p>
Heading 1 <h1> </h1>
Division <div> </div>
Bold Text <b> </b>
Italic Text <i> </i>
Underline <u> </u>
Table Row <tr> </tr>
Table Data <td> </td>
Link <a> </a>
List Item <li> </li>

ATTRIBUTES

HTML attributes provide additional information about HTML elements. They are always specified in the start tag and usually come in name/value pairs like: name="value"

  • href - Specifies the URL of the page the link goes to
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for an image
  • class - Specifies one or more class names for an element

HEADINGS AND PARAGRAPHS

Headings and paragraphs are used to organize and display text properly on a webpage. HTML provides six levels of headings, from h1 to h6 where ,h1 is the largest and most important heading, usually used for titles, and h6 is the smallest. Headings help structure the content and make it easier to read. The p tag is used to define a paragraph of text. It automatically adds space before and after the text, making the content look clean and readable. Together, headings and paragraphs make a webpage well-organized and easy to understand.

LISTS

HTML lists are used to display information in an organized way. There are three main types of lists in HTML — ordered lists, unordered lists, and definition lists. An ordered list (ol) shows items in a numbered format, like 1, 2, 3. An unordered list (ul) displays items with bullet points. Each item inside these lists is written using the (li) tag, which stands for “list item.” The definition list (dl) is used for terms and their meanings — it contains (dt)for the term and (dd) for its description. Lists make content neat, structured, and easier to read on a webpage.

COMMENTS

Comments in HTML are used to add notes or explanations within the code that are not shown on the webpage. They help developers understand the structure or purpose of the code. A comment starts with <!-- and ends with -->. Anything written between these symbols will be ignored by the browser. For example: <!-- This is a comment -->. Comments are very useful for organizing code, adding reminders, or explaining sections of an HTML document for future reference.
