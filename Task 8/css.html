INTRODUCTION TO CSS

BASICS OF CSS

CSS (Cascading Style Sheets) is used to style and design web pages. It defines how HTML elements should look — including colors, fonts, spacing, and layout. It helps separate the structure (HTML) from the presentation (CSS), making websites easier to maintain and more visually attractive.

There are three ways to use CSS:

  • Inline CSS: Written directly in the HTML tag using the style attribute.
  • Internal CSS: Written inside the <style> tag in the HTML document.
  • External CSS: Written in a separate .css file and linked to the HTML using <link> tag.

Among these, external CSS is preferred because it keeps design and structure separate, allowing easy updates across multiple web pages.

SELECTORS AND PROPERTIES

Selectors are patterns used to target HTML elements for styling. CSS properties define what style to apply. Each property has a value that specifies how it should behave.

Example: p { color: blue; font-size: 18px; }

Common selectors include:

  • Element selector: Targets all elements of a type
  • Class selector: Targets elements with a specific class
  • ID selector: Targets a unique element
  • Universal selector: Uses to target all elements.
  • Group selector: Combines multiple selectors

COLORS AND BACKGROUNDS

CSS supports multiple ways to define colors such as names (red), HEX codes (#ff0000), RGB (rgb(255,0,0)), and HSL (hsl(0,100%,50%))).

Backgrounds can include solid colors, gradients, or images:

  • background-color: Sets a solid background color.
  • background-image: Adds an image as a background.
  • background-repeat: Controls if the image repeats.
  • background-size: Adjusts how the image fits the area.
  • background: linear-gradient(...) creates smooth color transitions.

BORDERS AND MARGINS

Borders are lines around elements. CSS supports many styles: solid, dashed, dotted, double, groove, and ridge.

  • border-style: Defines type (solid, dashed, etc.).
  • border-width: Sets thickness.
  • border-color: Defines color.
  • border-radius: Rounds corners of the element.

Margins create space outside elements, while padding creates space inside. Together, they help control layout spacing.

CSS BOX MODEL

Every element in HTML is considered a rectangular box made up of four parts:

  • Content: The actual text or image inside the element.
  • Padding: The space between content and the border.
  • Border: The outer edge surrounding padding and content.
  • Margin: The space outside the border that separates elements.

Understanding the box model helps in accurate alignment and spacing control on web pages.

TRANSITIONS AND ANIMATIONS

CSS transitions and animations make web pages look dynamic and interactive. They allow smooth changes between styles over time.

  • transition-property: The property to animate.
  • transition-duration: The time the transition takes.
  • transition-timing-function: The speed curve of the transition.
  • animation: Used with keyframes to create complex motion effects.
